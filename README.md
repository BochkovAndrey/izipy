Окружение разработки end-to-end решения для автоматического сбора, хранения и визуализации данных из API.
Стек: Docker, Airflow, PostgreSQL, Redis, Prometheus (+ Blackbox/StatsD/node exporter/прочие), Grafana, Superset, REST API.
DevOps-инфраструктура с изоляцией сетей, подготовленное к деплою на VDS. Настроены службы мониторинга Prometheus + Grafana + Alertmanager с отправкой алертов в webhook бота Pachca.
